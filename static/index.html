Jalan Jalin API

Welcome to Jalan Jalin API, a simple greeting API for
Halal Bihalal Keluarga Besar Abdullah Salam 2024.

API Endpoint

GET /all_greetings

Get a simple greeting message from Jalan Jalin API.

POST /add_greeting?name=YOUR_NAME&greeting=YOUR_GREETING

Get a simple greeting message from Jalan Jalin API with a specific name and greeting.

POST /clear_greetings

Clear all greeting messages from Jalan Jalin API.
